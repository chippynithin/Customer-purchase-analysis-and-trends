# **Project Documentation**
## **Project Title: Customer Purchase Trends and Analysis**

### 1.	Project Overview
   
This project focuses on analysing sales data using Pandas for data cleaning and analysis, and Matplotlib, Seaborn and Plotly for data visualization. The goal is to understand customer purchasing behavior, product preferences, spending patterns, seasonal trends, and churn risk inorder to support better business decisions.

### 2.	Tools Used
   
•	Pandas – Data cleaning and manipulation

•	Matplotlib – Basic data visualization

•	Seaborn – Advanced and statistical visualizations

•	Plotly     - Interactive visualizations

### 3.	Dataset
   
•	Source: Mendeley data

•	Description: The dataset contains structured sales information with the following key columns:

o	Customer ID

o	Age

o	Gender

o	Item Purchased

o	Category

o	Purchase Amount

o	Location 

o	Size

o	Color

o	Season

o	Review Rating

o	Subscription Status

o	Payment method

o	Shipping Type

o	Discount Applied

o	Promo code used

o	Previous purchases

o	Preferred payment method

o	Frequency of purchases

### 4.	Steps Followed
   
   Imported the dataset using Pandas.
   
   Checked for missing values,duplicates and other inconsistencies
   
   Performed exploratory data analysis (EDA) using Pandas.
   
   Created visualizations using Matplotlib, Seaborn and Plotly, such as pie charts, histograms, Line charts ,heatmaps, scatter plots, stacked bar charts, count plot, box plot
   
   Interpreted the results to extract meaningful insights.



### 5.	Key Insights

o	Clothing and accessories are the highest performing product categories.

o	Medium and large size account for majority of purchases

o	Customer age and purchase history analysis shows a diverse customer base.

o	Males contribute to 68% of revenue.

o	Loyal, VIP and repeat customers contribute significantly to revenue

o	Summer is the worst season in terms of revenue and orders.

o	High spender customers giving low ratings is a potential churn risk.

### 6.	Visualizations
   
•	Line chart showing seasonal sales trends

•	Pie charts for percentage distribution.

•	Heatmap displaying correlation between season, category, size and age groups.

•	Stacked bar charts, count plots, scatter plots, histograms and box plot.
